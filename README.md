## 📌 Project Overview
This project analyzes African climate trends, focusing on temperature changes, rainfall variability, and climate extremes across different regions.

The analysis aims to understand how climate change is affecting Africa’s environment, agriculture, water systems, and livelihoods.

---

## 🎯 Objectives
- Analyze temperature trends across Africa  
- Study rainfall and hydro-climate variability  
- Identify climate extremes and impacts  
- Highlight regional climate risks  
- Propose adaptation and early warning solutions
